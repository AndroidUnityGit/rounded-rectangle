# rounded-rectangle
unity shader 正方形 长方形 切圆角

1.创建一个材质


2.将yy.shader 放上去


3.填写宽高比，（注意：如果宽高比<1, 则Radius 滑动条只能拉到一半，剩下的一半没用）
